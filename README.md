# Snake (C++ Console)

## Description
**Snake (C++ Console)** — это консольная реализация классической игры «Змейка» с использованием объектно-ориентированного подхода.  

Игрок управляет змейкой, которая движется по полю, собирает еду и растёт в длину. Цель — набрать максимальное количество очков, избегая столкновений со стенами и своим хвостом.  

Проект демонстрирует навыки:
- Классы и инкапсуляция  
- Работа с векторами (std::vector) для динамического хвоста  
- Использование перечислений (enum) для направления движения  
- Организация игрового цикла и обработка ввода с клавиатуры  
- Простая система коллизий  

## Features
- Движение змейки с клавишами 'W', 'A', 'S', 'D'  
- Рост змейки после съедания еды ('+')  
- Рандомная генерация еды  
- Проверка столкновений со стенами и хвостом  
- Подсчет очков  
- Сообщение 'Game Over' при поражении  

## Technologies
- C++  
- STL (`vector`)  
- Windows API (`windows.h`) для управления курсором  
- Консольный вывод
